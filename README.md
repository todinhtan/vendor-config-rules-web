# Vendor rules config page #

## Features
* Update vendor's rules

## Development
#### Install dependencies
```bash
npm install
```
#### Run app 
```bash
npm run dev
```
#### or watch changes
```bash
npm run watch
```

## Production
#### Install dependencies
```bash
npm install
```
#### Build dist
```bash
npm run build
```
#### Run
```bash
npm run start
```
